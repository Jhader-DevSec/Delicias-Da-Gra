# 🍰 Delícias da Gra - Landing Page

Repositório dedicado ao desenvolvimento de uma landing page moderna e responsiva para a confeitaria e lanchonete "Delícias da Gra". O projeto foi construído do zero para aprimorar fundamentos de **Frontend**, UI/UX e manipulação avançada do DOM sem o uso de frameworks.

## 🚀 Tecnologias e Conceitos Aplicados

`HTML5 Semântico` `CSS3 Avançado` `Vanilla JavaScript` `Flexbox` `CSS Masks & Gradients` `DOM Manipulation` `Responsive Design`

## ✨ Destaques e Funcionalidades

O layout foi pensado para ser atrativo e focado na conversão de pedidos via WhatsApp, contando com os seguintes destaques técnicos:

* **Carrosséis Autônomos Bidirecionais:** Desenvolvidos com Vanilla JS, os carrosséis de produtos rolam automaticamente em direções opostas (loop contínuo) e pausam a animação no evento de `hover` do usuário.
* **Efeitos Visuais Avançados (CSS Masks):** Utilização de `radial-gradient` e `-webkit-mask-image` para criar o efeito de esfumaçamento e mesclagem das imagens de destaque com o fundo.
* **Navegação Suave (Smooth Scroll):** Implementação nativa via CSS para rolagem fluida entre as seções a partir do menu de navegação.
* **Integração de Mapas:** Iframe nativo e responsivo do Google Maps apontando para o endereço físico do estabelecimento.
* **Mobile-Friendly:** Adaptação completa do layout para dispositivos móveis utilizando Media Queries.
